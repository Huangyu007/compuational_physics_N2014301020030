## 作业完成步骤与截图
#### 第一步：建立一个循环，次数为50次
#### 第二步：在循环中插入自己的名字图样运行即可。同时中间要插入清屏代码
#### 平移代码
```python
import os
import time

for i in range(50):
    a=(" "*i)
    print (a+"*********        **********")
    print (a+"**********       **********")
    print (a+"**      **            **")
    print (a+"**      **            **")
    print (a+"**      **            **")
    print (a+"********              **")
    print (a+"******                **")
    print (a+"**                    **")
    print (a+"**                    **")
    print (a+"**                    **")
    print (a+"**                   **")
    print (a+"**                ****")
    print (a+"                      ")
    time.sleep(0.1)
    os.system('cls')
